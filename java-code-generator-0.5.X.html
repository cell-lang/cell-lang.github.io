Cell ➞ Java compiler version 0.5.2

Version 0.5.2 of the Cell to Java code generator is now available. Apart from a number of under-the-hood architectural improvements it brings an expanded standard library and performance improvements for relational automata.

Version 0.5.2 change log

  • Added date, time, sorting and UFT-8 functions to the standard library
  • Improved performance of relational automata

Version 0.5.1 change log

  • Fixed bug that caused the compiler to generated, in some specific circumstances, Java code that would not compile
  • Improved performance of relational automata
